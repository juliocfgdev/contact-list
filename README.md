# contact-list
## Descrição
Aplicação de lista de contatos feita para treinar programação orientada à objeto usando sintáxe tanto do ES5 quanto do ES6
### Tecnologias Usadas:
* HTML5
* CSS3 + Skeleton CSS
* Vannila JS + ES6:
  * Arrow Functions
  * OOP sem classes
  * OOP com Classes
* GitHub API
#### Live Version: https://juliocfgdev.github.io/contact-list/
